# Bettamind Brand Source

Place the owner-approved canonical Bettamind logo in this folder:

- preferred: `brand/source/bettamind-logo-master.svg`
- fallback: `brand/source/bettamind-logo-master.png`

Rules:

- Do not overwrite the source logo.
- Do not recolour the logo until Phase 2 selects and documents one accessible
  Bettamind palette.
- Keep generated previews under `brand/generated/`.
- Copy only approved generated assets into Android and iOS platform catalogues.

Phase 1 does not generate final icons or brand variants.
