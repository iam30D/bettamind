# Manual Owner Checklist

## Before Phase 2

- Add the canonical Bettamind logo to `brand/source/`.
- Confirm whether the source SVG lettering is converted to paths or provide
  the relevant font licence.
- Review and approve Phase 1 build foundation.
- Run the Codemagic unsigned iOS simulator workflow.
- Share the first meaningful Codemagic error if the workflow fails.

## Before release phases

- Confirm final Android application ID.
- Confirm final iOS bundle identifier.
- Join the Apple Developer Program.
- Configure Apple signing only in Codemagic secure credentials.
- Never commit signing files or credentials.
