# Bettamind Product Report

## Product purpose

Bettamind is a private, mostly offline Human Growth Operating System for adults
aged 18 and above. It guides users through awareness, choice, action,
consequence, reflection, repair and growth.

Bettamind is not a chatbot, therapy product, diagnostic product, legal service,
financial adviser or emergency service.

## Core values

- dignity;
- truth-seeking;
- compassionate love;
- justice;
- autonomy;
- responsibility;
- humility;
- repair.

## Locked product boundaries

- Core use works offline.
- Core use requires no account.
- Core use works without a backend.
- Core use works without AI.
- The optional backend must never be necessary for core use.
- AI models are optional, local, replaceable and removable.
- Personal content never leaves the device without purpose-specific consent.
- Permanent memory requires separate user approval.
- No advertising, public feeds, public rankings, human-worth scores or manipulative streaks.

## Adult-only MVP

The MVP is for adults aged 18 and above. Age assurance uses available Apple or
Google age-range signals when possible and falls back to local self-declaration.
The MVP must not request identity documents or store exact dates of birth.

If the user is under 18 or unknown, adult personal features are not initialised.

## Privacy and safety

Bettamind defaults to local-only operation. Optional sync, exports, support
sharing, AI model downloads and backend use require separate purpose-specific
consent.

Safety-sensitive flows must use deterministic safeguards first. AI must never
be the only safety layer.

## Global audience

English is the source locale. The architecture must support BCP 47 locale tags,
RTL layout, locale-aware dates/numbers/plurals and script-aware font fallback.
Initial target locales are English, French, Spanish, Portuguese, Arabic, Hindi,
Simplified Chinese, Hausa, Yoruba and Igbo.
